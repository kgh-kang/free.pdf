# free.pdf

브라우저에서 PDF를 편집하세요. 서버 전송 없이 모든 작업이 로컬에서 처리됩니다.

## 기능

### PDF 편집
- 여러 PDF 합치기
- 페이지 추출 / 삭제
- 페이지 순서 변경 (드래그)
- 페이지 회전
- 파일별 컬러 구분
- 원본 크기 미리보기

### 이미지 변환
- 이미지(JPG, PNG) → PDF 변환
- PDF → 이미지(PNG, JPG) 변환

### 텍스트 추출
- PDF에서 텍스트 복사
- TXT 파일 다운로드

### 워터마크
- 텍스트 워터마크 (대각선, 가로, 타일)
- 이미지 워터마크 (서명/도장)
- 페이지 번호 추가

## 특징

- **서버 전송 없음** — 파일이 브라우저를 떠나지 않습니다
- **완전 무료** — 광고 없음, 숨겨진 결제 없음
- **회원가입 불필요** — 접속하면 바로 사용
- **설치 불필요** — 브라우저만 있으면 OK

## 기술

- HTML5 + Vanilla CSS + Vanilla JS
- [pdf-lib](https://pdf-lib.js.org/) — PDF 생성/편집
- [pdf.js](https://mozilla.github.io/pdf.js/) — PDF 렌더링/텍스트 추출
- [Pretendard](https://github.com/orioncactus/pretendard) — 본문 폰트

## 개인정보

모든 작업은 브라우저에서만 처리됩니다. 파일이 외부 서버로 전송되지 않습니다.
