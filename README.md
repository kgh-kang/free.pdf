# 진짜무료PDF

PDF 합치기 & 텍스트 추출 도구

## 특징

- **서버 전송 없음** — 모든 작업이 브라우저에서만 처리됩니다
- **완전 무료** — 광고 없음, 숨겨진 결제 없음
- **회원가입 불필요** — 접속하면 바로 사용
- **설치 불필요** — 브라우저만 있으면 OK

## 기능

### PDF 합치기
- 여러 PDF 파일을 하나로 합칩니다
- 드래그 앤 드롭으로 파일 추가
- 드래그로 순서 변경
- 파일별 페이지 수, 용량 확인

### 텍스트 추출
- PDF에서 텍스트를 추출합니다
- 클립보드 복사 / TXT 파일 다운로드
- 이미지 기반 PDF는 지원하지 않습니다

## 사용 기술

- [pdf-lib](https://pdf-lib.js.org/) — PDF 합치기
- [pdf.js](https://mozilla.github.io/pdf.js/) — 텍스트 추출

## 개인정보

파일은 외부 서버로 전송되지 않습니다. 모든 처리는 사용자의 브라우저에서 이루어집니다.
